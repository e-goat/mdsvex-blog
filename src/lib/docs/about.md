Hi, my name is Martin Duchev and I love to mess around with tech. Currently, I am focused on building reliable, maintainable web applications. My approach prioritizes clean architecture, practical solutions, and code that other developers can work with confidently—in general, code that lasts.

## Tech Stack

#### **Backend**
1. **PHP** - This is where I was forged as a developer. I'm not going to say that I love **Laravel** (been there since **Laravel 5**).
2. **Node** - I've used it every once in a while.
3. **Deno** - Ryan Dahl did a great job! What a well-curated JavaScript runtime.
4. **Bun** - I've barely touched the surface here, but it seems that people like it. I don't think I've liked it since the last AI rewrite at Anthropic Labs.
5. **Python** (Basic) - Used it mostly imperatively for inline statements & loops.
6. **NextJS** - Great idea, one of the first full-stack JS kits out there. Absolutely devastating developer environment considering Vercel, v0, and their vast selection of templates in the picture. However, it grew to an extent where a simple 'hello world' application weighs more than your mum (kek).
7. **Drizzle ORM** - It's OKAY.
8. **Prisma** - Their team did a great job with features like Queries or Accelerated. The community is expanding. In general, using [Prisma](https://www.prisma.io) is a pleasant experience + it's very well integrated with [Deno Deploy](https://deno.com/deploy).

#### **Frontend**
1. **JavaScript** - The mighty, the one above them all.
2. **React** - Not a huge fan. Mainly use it when I deal with NextJS applications, nothing much I can say about it. It's the leading framework on the market.
3. **Svelte/SvelteKit** - My go-to framework when it comes to JavaScript integrations. This blog is currently running on top of SvelteKit. It's lightweight, it's fast, and it's growing a healthy ecosystem. Tools like [mdsvex](https://mdsvex.pngwn.io/) or [Skeleton](https://www.skeleton.dev/) make it stand out in the JS market warfare. The community is growing as well, and people there are proficient and very helpful. Agents are starting to understand Svelte more and more (wasn't the case before).
4. **Tailwind CSS** - This is my preferred stylization tool.

#### **AI Tools**
1. **Copilot CLI**
2. **Claude CLI**
3. **Cursor** - Used it in the early days while it was just a fork of VSCode. Great idea IMO.
4. **Zed** - Currently a Zed user; used it extensively with Zed ACPs (Came for the AI integration, stayed for the ease of use and performance).
